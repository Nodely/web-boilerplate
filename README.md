## Getting Started

Clone repo:

`git clone https://github.com/Nodely/web-boilerplate`

Install dependencies

`cd web-boilerplate && npm install`

Launch local version

`cd web-boilerplate && npm start`

Your web site will be available by `http://localhost:3000`
